### 계좌정보와 가계부 프로그램(WPF ver)<hr/>

#### 1. 주요기능(미완성)
##### ● SQL 연동하여 Data Handling
##### ● DataBase를 활용하여 계좌정보 및 가계부정보 출력(추가, 삭제, 수정)
##### ● ChartControl을 이용하여 선택한 정보의 그래프 출력 기능
##### ● 날짜에 따른 가계부 조회 가능 
##### ● 국가별 계좌 정보 분류 가능 <hr/>
